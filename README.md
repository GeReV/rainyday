## RainyDay Screensaver

RainyDay is a Windows screensaver inspired by [RainyDay.js](https://github.com/mubaidr/rainyday.js) by [Muhammad Ubaid Raza](https://github.com/mubaidr).

It is written in Rust and OpenGL.
